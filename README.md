# Week 04 · Day 22 (Session) — Assignment

This repository contains the solution for the **PG Diploma · AI-ML & Agentic AI Engineering · IIT Gandhinagar** assignment on Week 4, Day 22.

## Topics Covered
- **Pandas**: Selection with `loc`, `iloc`, filtering multiple conditions, and descriptive statistics (`describe()`).
- **Matplotlib**: Creating visualizations such as Histograms, Bar Plots, Line Charts, and KDE Plots.

## Project Structure

- `sales_data.csv`: A synthetic dataset with 600 rows of transaction data used for all tasks.
- `part-a-am.py`: Implements Part A — Concept Application. Includes selection, filtering, summary statistics, and four different visualization types.
- `part-b-am.py`: Implements Part B — Stretch Problem. Features grouped analysis by region and comparison of numerical features (Sales vs. Profit).
- `part-c-am.py`: Implements Part C — Interview Ready. Contains conceptual answers and coding examples.
- `part-d-am.py`: Implements Part D — AI-Augmented Task. Documents the AI-generated explanation and evaluation.

## Prerequisites
- Python 3.x
- Pandas
- Matplotlib
- Seaborn (used for KDE plots)

## How to Run
Follow these steps to run the assignment parts:

```bash
# Install dependencies if you don't have them
pip install pandas matplotlib seaborn

# Run individual parts
python part-a-am.py
python part-b-am.py
python part-c-am.py
python part-d-am.py
```

## Insights
- **Region-based Analysis**: South region showed slightly higher average profit compared to others in this dataset.
- **Distribution**: The sales distribution is fairly uniform, while the profit distribution is concentrated in lower ranges showing the profit-to-sales ratio.
- **Visual Comparison**: KDE plots provide a smoother perspective on probability density compared to standard Histograms.
